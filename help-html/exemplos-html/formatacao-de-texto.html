Um trecho de código: alert("Olá mundo!")
Código:
1 Um trecho de código: <code>alert("Olá mundo!")</code>.
2 <!-- a tag <code> indica um trecho de código inline, ela também aplica uma fonte monoespacaçada, o que é ideal para a leitura de códigos --> 
let nome = prompt("Informe seu nome: ")
let sexo = prompt("Informe seu sexo (F/M):").toUpperCase()
alert( 
	sexo == "M" ? `Bom dia Sr. ${nome}` : `Bom dia Sra. ${nome}` 
)
Código:
1 <pre>
2 let nome = prompt("Informe seu nome: ")
3 let sexo = prompt("Informe seu sexo (F/M):").toUpperCase()
4 alert(
5 sexo == "M" ? `Bom dia Sr. ${nome}` : `Bom dia Sra. ${nome}`
6 )
7 </pre>
8 <!-- A tag <pre> é usada para inserir textos pré formatados, ou seja ela respeita todos espaços quebras de linha inseridos dentro dela. -->
