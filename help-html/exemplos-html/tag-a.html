Código:
1 <a href="../index.html">Tabela HTML</a><br>
2 Uma ligação comum.
3 <!-- A tag <a> cria uma ligação com outro documento por meio de um hyperlink -->
4 <!-- O atributo href="" indica o caminho até o documento -->

Target para a tag <a>

Tabela HTML
Este link será aberto na mesma guia.
Código:
1 <a href="../index.html" target=_self>Tabela HTML</a>

Exemplos:

_self abre o link na mesma guia (padrão).
_black abre o link em nova guia.
