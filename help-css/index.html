Propriedades para Bordas

border-width:
Define a expessura das bordas.
border-top-width:
Define a expessura da borda de cima.
border-left-width:
Define a expessura da borda direita.
border-bottom-width:
Define a expessura da borda de baixo.
border-right-width:
Define a expessura da borda esquerda.

border-color:
Define a cor das bordas.
border-top-color:
Define a cor da borda de cima.
border-left-color:
Define a cor da borda esquerda.
border-bottom-color:
Define a cor da borda de baixo.
border-right-color:
Define a cor da borda direita.

border-style:
Define o estilo das bordas.
border-top-style:
Define apenas o estilo da borda de cima.
border-left-style:
Define apenas o estilo da borda esquerda.
border-bottom-style:
Define apenas o estilo da borda de baixo.
border-right-style:
Define apenas o estilo da borda direita.

border:
Define width, color e style de todas as bordas.
border-top:
Define width, color e style da borda de cima.
border-right:
Define width, color e style da borda direita.
border-bottom:
Define width, color e style da borda de baixo.
border-left:
Define width, color e style da borda esquerda.

border-radius:
Arredonda as bordas.
border-top-left-radius:
Arredonda apenas a borda superior esquerda.
border-top-right-radius:
Arredonda apenas a borda superior direita.
border-bottom-left-radius:
Arredonda apenas a borda inferior esquerda.
border-bottom-right-radius:
Arredonda apenas a borda inferior direita.

Propriedades para Texto

text-align:
Define o alinhamento do texto.

text-decoration-line:
Define o tipo linha decorativa.
text-decoration-color:
Define a cor da linha decorativa.
text-decoration-style:
Define o estilo da linha decorativa.
text-decoration:
Atalho para as propriedades acima.

font-size:
Define o tamanho da fonte.
font-weight:
Define a tonalidade da fonte.
font-style:
Define o estilo da fonte(itálico por exemplo).
font-family:
Define a família da fonte.

Propriedades para Listas

list-style-type:
Define o estilo da lista.
list-style-image:
Define uma imagem para enumerar a lista
list-style-position:
Define a posição do estilo da lista
